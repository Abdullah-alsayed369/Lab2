# Lab 2: Video Game Sales Analysis

### 1. Dataset Description
The dataset used is **vgsales.csv**, which contains a list of video games with sales greater than 100,000 copies. It includes features like Platform, Year, Genre, and Publisher.

### 2. Machine Learning Problem
* **Type:** Regression
* **Target Variable:** `Global_Sales`
* **Goal:** To predict the total global sales of a video game based on its release features (Genre, Platform, etc.).
